Convenience class to help with handling Close Channel notifications

Create an instance with the intent or remoteMessage from a push notification and use the returned CloseChannelNotification to either create new (internal) notification or send notification or use it to open the correct channel see Companion for static functions to create this helper class

Sample Usage for receiving remote message when running app

val receivedCloseNotification = CloseChannelNotification.from(remoteMessage)
if (receivedCloseNotification != null) {...}
